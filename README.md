# Lesson 1.1 – Min hobby (text)

I detta steg jobbar vi med **en enda ny sida** och fokus ligger på text.

Den nya sidan heter:
- **Min hobby** (`hobby.html`)

Syftet är att:
- skriva sammanhängande text
- använda rubriker och stycken
- beskriva en sak i taget

Här introducerar vi **inga listor och inga tabeller** ännu.
Det kommer i senare steg.

---

## Hitta till den nya sidan

Just nu finns **ingen meny** på hemsidan.  
För att komma till sidan *Min hobby* gör du så här:

1. Gå till startsidan (`index.html`)
2. Lägg till `/hobby.html` sist i webbadressen  
   Exempel:  
   `https://DITT-NAMN.github.io/DITT-REPO-NAMN/hobby.html`
3. Tryck **Enter**

Du kan också:
- öppna filen `hobby.html` direkt i GitHub
- klicka på **View / Open in browser** om det visas

---

## Uppgift
1. Öppna `hobby.html`
2. Skriv om en hobby eller ett intresse du har
3. Använd:
   - en rubrik (`<h1>`)
   - flera stycken (`<p>`)
4. Kontrollera att länken tillbaka till startsidan fungerar

Ta god tid på dig och fokusera på innehållet.
Du ska inte ta bort något du gjort tidigare.
